**Nicho exato:** Não somos um app de horóscopo diário. Somos uma **experiência ritual de autoconhecimento** com IA — seis tradições filosóficas reais, mapa irrepetível, oráculo personalizado. Concorrentes fazem horóscopo. Nós fazemos cartografia.

**Diferencial competitivo:**
1. **Profundidade** — 6 sistemas cruzados (não só astrologia)
2. **Personalização** — bridges cross-system, body variations, carta randômica
3. **Experiência ritual** — cerimônia de abertura, som, geometria sagrada
4. **Proveniência** — cada conclusão rastreável (ninguém faz isso)
5. **Soul Mate com score** — compatibility % + match/mismatch (viral)

## Modelo de Receita

### Tiers (freemium progressivo)

```
TEASER · R$0 · Sem email
  ├── Cartas 1-2 completas (astrologia + kabbalah)
  ├── Carta 3: abertura + micro-revelação da sombra (1 frase)
  ├── Mapa final resumido (signo + arquétipo + número)
  └── CTA: "sua sombra já apareceu. para ver o que ela revela..."

REVELAÇÃO · R$0 · Com email
  ├── Journey completa (5 cartas)
  ├── Oráculo: 1 pergunta por carta
  ├── Mapa final completo com síntese
  ├── Link salvo 30 dias
  └── Email com link do mapa + 3 emails de nurture

ORÁCULO · R$19 (~$3.50 USD) · Pagamento único
  ├── 3 perguntas por carta (15 total)
  ├── Soul Mate disponível
  ├── Link permanente 12 meses
  ├── Export PNG/PDF
  └── Email de ativação + onboarding

SOUL MATE · R$29 (~$5.50 USD) · Add-on
  ├── Dois mapas cruzados com score %
  ├── Match/mismatch detalhado
  ├── Oráculo do encontro (3 perguntas)
  └── Link permanente compartilhável

VITALÍCIO · R$97 (~$18 USD) · One-time
  ├── Todas as leituras ilimitadas
  ├── Soul Mates ilimitados
  ├── Oráculo sem limite
  ├── Histórico de mapas
  └── Acesso antecipado a novos sistemas
```

### Por que pagamento único (não subscription)

1. **Conversão mais alta** — LATAM resiste a assinaturas recorrentes
2. **Valor percebido** — "meu mapa" é um artefato, não um serviço mensal
3. **Simplicidade** — sem churn, sem cancelamento, sem billing disputes
4. **Upsell natural** — Soul Mate é produto separado, Vitalício é upgrade

### Por que esses preços

- R$19 = preço de uma consulta de tarot online (referência cultural BR)
- R$29 = Soul Mate é "para dois" — cada um paga ~R$15
- R$97 = preço de um curso básico online — âncora psicológica
- Preço internacional: usar purchasing power parity (PPP)

## Internacionalização

### Fase 1: Português (BR) — já está

### Fase 2: Inglês (US/UK/AU)
- **Prioridade:** maior mercado global de wellness apps
- **Pricing:** $3.99 / $5.99 / $19.99 (PPP ajustado)
- **Localização:** traduzir engine content (variações, descriptions, shadows)
- **CTA cultural:** "discover your shadow" (Jung ressoa em EN)
- **Canais:** Reddit r/astrology (2M), Instagram, TikTok mystical community

### Fase 3: Espanhol (LATAM)
- **Prioridade:** segundo mercado mais próximo, culturalmente alinhado
- **Pricing:** R$19 equivalente local (MXN, ARS, COP)
- **Localização:** conteúdo traduzido + variações culturais
- **Canais:** Instagram, TikTok, comunidades espirituais hispanas

### Fase 4: Francês / Italiano / Alemão
- **Europa continental tem forte demanda por wellness espiritual**
- **Pricing:** €4.99 / €7.99 / €19.99

### Automação da tradução
```
1. Engine texts (engine/*.ts) → arquivo i18n JSON por idioma
2. API prompts (api/carta.ts, api/oracle-carta.ts) → system prompt localizado
3. UI labels → i18n hook com lazy loading por locale
4. URL: cartografia.app/en, cartografia.app/es, etc.
5. OG tags dinâmicas por idioma
```

## Princípios do Funil

1. **Micro-revelação vende.** A frase que a pessoa reconhece em si é o que converte.
2. **Email é o ativo.** Não é newsletter — é canal de retorno + identidade.
3. **Soul Mate é o loop viral.** Cada leitura gera convite para outra pessoa.
4. **Carta randômica é a retenção.** Razão para voltar sem precisar de conteúdo novo.
5. **Automação total.** Zero intervenção humana do click ao pagamento.
6. **Localização multiplica.** Mesmo produto, mercados diferentes, receita multiplicada.
