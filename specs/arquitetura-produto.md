# Arquitetura de Produto

*E. — 04 Abr 2026*

---

## Dois produtos. Completos em si mesmos.

**Cartografia da Alma** existe independente.
**MindRoot** existe independente.

Um não precisa do outro.
Quando se encontram, algo maior emerge —
mas nenhum depende do outro para ter valor completo.

---

## Cartografia da Alma

Produto standalone.

Responde: *quem você é.*

- SoulMap completo (signo, sephirah, arquétipo, frequência, numerologia, palma)
- Oráculo — 3 perguntas com contexto real
- Soul Mate — dois mapas cruzados
- Monetização própria (R$19 leitura / R$29 Soul Mate)
- Deploy próprio (cartografia.com.br ou similar)
- Público: qualquer pessoa — com ou sem MindRoot

A pessoa pode usar o Cartografia a vida inteira
sem saber que o MindRoot existe.

---

## MindRoot

Produto standalone.

Responde: *como você vive.*

- Captura ação, emoção, tempo, alma
- Organiza a vida com intenção
- Deploy próprio (mindroot.com.au)
- Público: qualquer pessoa — com ou sem Cartografia

A pessoa pode usar o MindRoot a vida inteira
sem saber que o Cartografia existe.

---

## O que compartilham

Infraestrutura, não dependência.

| Camada | Compartilhado |
|--------|--------------|
| Database | Supabase — mesma instância, schemas separados |
| Auth | Mesma conta — uma vez só |
| Documento legal | Privacidade e termos unificados |
| Design system | Atom HS — mesma linguagem visual |
| Genesis | Schema v5.0.1 — AtomItem como envelope |

O SoulMap é um AtomItem com `type: reading`.
Ele existe na DB compartilhada.
O MindRoot pode lê-lo — se o usuário tiver os dois.
Mas o Cartografia não depende do MindRoot para funcionar.
E o MindRoot não depende do Cartografia.

---

## Quando se encontram

Se a pessoa tem os dois:

- SoulMap já existe quando entra no MindRoot
- Não precisa recadastrar nada
- Correlação começa imediatamente:
  dados reais de vida × camadas ocultas do mapa
- O Cartografia vira porta de entrada natural para o MindRoot

Mas isso é consequência — não arquitetura forçada.

---

## O Cartografia como porta de entrada

Não é funil. É jornada natural.

A pessoa que quer entender quem é
é exatamente a pessoa que vai querer organizar
como vive com intenção.

O Cartografia não diz "vem pro MindRoot."
Ele simplesmente existe.
E quando a pessoa chega no MindRoot,
o mapa já está lá.

---

## Analogia

É a mesma lógica do Soul Mate.

Cada mapa existe sozinho.
O cruzamento é opcional.
O espaço entre emerge quando os dois escolhem se encontrar.

Nenhum mapa precisa do outro para ser inteiro.
